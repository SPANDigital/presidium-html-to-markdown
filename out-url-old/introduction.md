ACI Kafka is a fully hosted and managed [Apache Kafka](https://kafka.apache.org/)
service that’s secure, highly available, reliable, and easy to use.

Start with an [Overview]({{< ref "overview" >}}) of ACI Kafka if you’d like to
learn more about our service offering, existing features, project road
map, and service availability. If you’re interested in learning more
about on-boarding then visit our [On-boarding & Expansion]({{< ref "onboarding" >}})
page. If you have any questions not answered in our docs, don’t hesitate
to [contact us]({{< ref "contact" >}}).
