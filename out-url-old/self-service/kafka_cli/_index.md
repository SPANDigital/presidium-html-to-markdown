The Kafka CLI enables users to quickly modify their entities using a command-line interface. It is built on top of the [public API]({{< ref "self-service/public_api" >}}) using [OpenAPI Generator]({{%baseurl%}}/https://openapi-generator.tech). The CLI is published under the [Kafka CLI package](https://engweb.apple.com/pypi/pie-kafka-cli) with its [source available on PIE GitHub](https://github.pie.apple.com/pie/kafka-cli). The Python client requires Python 3.7 or later.
