The ACI Kafka team is excited to announce the release of ACI Kafka 0.12.
This release is based on Apache Kafka 1.0.1 and contains a number of new
features, fixes, and improvements.

The ACI Kafka 0.12.0 client is [available on\
Artifactory]({{%baseurl%}}//files/kafka-pie-client-0.12.0.jar)
and can be configured as a [Gradle\
Dependency]({{%baseurl%}}/../client/quick_start.html#gradle-dependency).
This version is recommended to all users.

Note that we have jumped from ACI Kafka 0.9 to ACI Kafka 0.12 to avoid
confusion with Apache Kafka releases.

For a full set of commits see: [the github compare\
view]({{%baseurl%}}/https://github.pie.apple.com/pie/kafka-tools/compare/release-0.9...release-0.12).

For the latest documentation, including an overview of ACI Kafka and the
ACI Kafka Client, please see visit our
[documentation](https://docs.aci.apple.com/kafka).
