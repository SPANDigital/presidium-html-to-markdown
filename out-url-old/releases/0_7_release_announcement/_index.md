The ACI Kafka team is excited to announce the release of ACI Kafka 0.7.
This release is based on Apache Kafka 0.10.2.1 and contains a number of
improvements and critical fixes to ACI Kafka Server and the ACI Kafka
and HubbleMetricReporter clients. This release also includes many new
foundational components which form the basis of PIE’s
Kafka-as-a-Service offering and is currently deployed to PIE’s QA and
production Kafka clusters.

Our documentation has also been updated with sample [Hubble Dashboard\
templates]({{< ref "monitoring/kafka_client_hubble_dashboards" >}})
to monitor your clients and instructions for onboarded ACI Kafka users
to [securely receive client\
credentials]({{< ref "client/client_secrets_distribution" >}}).

For the latest documentation, including instructions for using the ACI
Kafka client, please see visit our
[docs](https://docs.aci.apple.com/kafka).

An update for prospective customers: we’re currently planning FY18
onboarding and in the next few days we’ll be reaching out to
prospective customers we connected with in FY17 but have not yet
onboarded. If you have questions or would like to be considered for
