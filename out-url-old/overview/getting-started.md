Choose from the following options to help you get started using ACI Kafka:

- If you’re a new customer and ready to request ACI Kafka, start with [On-boarding & Expansion]({{< ref "onboarding" >}}).
- If you’re trying to decide if Apache Kafka is the right choice for your service, start with the [Apache Kafka documentation](https://kafka.apache.org/documentation/#uses).
- If you’re already familiar with Apache Kafka, but you’re not sure if ACI Kafka fits your use case, review the information below on this page for [Offerings](#offerings) and [ACI Kafka caveats and limitations](#aci-kafka-caveats-and-limitations).
- If you’re just starting to develop your service, you can use the [Development and CI]({{< ref "dev-and-ci" >}}) resources and the client configuration [Quick Start]({{< ref "client/quick_start" >}}) to help you with local development.
