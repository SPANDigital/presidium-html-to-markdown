weight: 4
---

Metrics available to consumer groups are enumerated below.

| Metric name | Description | Labels |
| --- | --- | --- |
| `kafka_consumergroup_group_lag` | The difference between the last produced offset and the last consumed offset for this partition in this topic partition for this consumer group. | `_ns_`, `_wc_`, `cluster_name`, `client`, `consumer_group`, `consumer_id`, `namespace`, `topic_name`, `partition` |
| `kafka_consumergroup_group_lag_seconds` | The estimated lag in seconds. This metric can be misleading if the producer is replaying old events with old timestamps. | `_ns_`, `_wc_`, `cluster_name`, `client`, `consumer_group`, `consumer_id`, `namespace`, `topic_name`, `partition` |
| `kafka_consumergroup_group_offset` | The last consumed offset for this partition in this topic partition for this consumer group. | `_ns_`, `_wc_`, `cluster_name`, `client`, `consumer_group`, `consumer_id`, `namespace`, `topic_name`, `partition` |
| `kafka_consumergroup_group_topic_sum_lag` | The sum of differences between the last produced offset and the last consumed offset of all partitions in this topic for this consumer group. Including idle consumer instances. | `_ns_`, `_wc_`, `cluster_name`, `client`, `consumer_group`, `namespace`, `topic_name` |
| `kafka_consumergroup_group_sum_lag` | The sum of the differences between the last produced offset and the last consumed offset of all partitions for this consumer group.Including idle consumer instances. | `_ns_`, `_wc_`, `cluster_name`, `client`, `consumer_group`, `namespace`, `topic_name` |

### Labels [¶](#labels "Link to this heading")

Each metric may include one of the following labels:

- `_wc_` : Mosaic workspace. This is a constant value set to `aci-kafka` and cannot be changed.
- `_ns_`: Mosaic namespace. Either `kafka-consumer-lag-prod` or `kafka-consumer-lag-if1`.
- `cluster_name`: Kafka cluster name.
- `consumer_group`: The value of [`group.id`]({{%baseurl%}}/consumer_client.html#consumer-group-a-k-a-group-id-config)
- `client`: ACI Kafka [client identity]({{%baseurl%}}/../concepts.html#client-identities). This value is extracted from `consumer_group`.
- `consumer_id`: The id of the consumer group member. Usually, this is automatically generated by the group coordinator. If the instance has been idle for 7 days, then `consumer_id` will be `unknown`.
- `namespace`: ACI Kafka [namespace]({{%baseurl%}}/../concepts.html#namespace).
- `topic_name`: ACI Kafka topic name. This is **not** the fully qualified name.
- `partition`: The Kafka partition.
