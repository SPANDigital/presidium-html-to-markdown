Warning

When checking connectivity to the brokers make sure you’re running the command from the same host, pod or instance that is logging the errors or timeouts.

[Synthetic Monitor dashboard](https://aci-kafka.grafana.telemetry.g.apple.com/d/d421ddc0-a18f-467f-93ae-3b71132999cf/synthetic-monitor) should display a command you can use to test connectivity to online brokers.

You can find out more about [testing and establishing connectivity in the dedicated guide]({{< ref "troubleshooting/connectivity" >}}).
