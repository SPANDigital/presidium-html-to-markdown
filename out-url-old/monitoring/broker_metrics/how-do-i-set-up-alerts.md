---
title: "How do I set up alerts?¶"
weight: 8
---

To set up alerting, you will need to:

1. On-board with [ACI Mosaic]({{%baseurl%}}//files/telemetry.g.apple.com).
2. Once you have your own Mosaic workspace, contact the ACI Kafka team and we will share the metrics with your workspace.
3. Once this is set up, follow [Mosaic’s alerting documentation](https://telemetry.g.apple.com/docs/alerting).
   If you have any other questions please contact the ACI Mosaic team directly on Slack in
   [`#help-aci-mosaic`](https://a1391190.slack.com/messages/CHNFM578E).

Note

[Synthetic metrics](#synthetic-metrics) are still under development and are not currently available for alerting. These can still be viewed in the [cluster metrics dashboard](https://aci-kafka.grafana.telemetry.g.apple.com/d/1uXsd--4k/cluster-dashboard-customer-only?orgId=1) for dedicated clusters.
