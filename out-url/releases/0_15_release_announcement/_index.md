title: "ACI Kafka 0.15 Release Announcement¶"
weight: 1
---

The ACI Kafka team is excited to announce the release of ACI Kafka 0.15.
This release is based on Apache Kafka 1.1.1 and contains a number of new
features, fixes, and improvements.

ACI Kafka 0.15 remains on Apache Kafka 1.1.1, the release version also
used in ACI Kafka 0.14, so no notable Apache Kafka changes are included.
Server binaries continue to use 1.1 protocol and log format versions in
all environments. See the [Internal Link: ACI Kafka 0.14 release
notes](0_14_release_announcement.html) for more information on Apache
Kafka 1.1.1 changes.

In the next ACI Kafka release, we will be upgrading to Apache Kafka 2.1,
and we expect protocol and log format version upgrades in this release.
At the end of April, we will also announce our new Public Restful API, a
language-agnostic API for automating the management of your resources.

For the latest documentation, including an overview of ACI Kafka and the
ACI Kafka Client, please see our
[External Link: documentation](https://docs.aci.apple.com/kafka).
