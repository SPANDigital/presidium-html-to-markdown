### FY2020 - Quarter 4[Internal Link: ¶](\#fy2020-quarter-4)

- **Automated Onboarding for Playgrounds** Providing fast, easy
  onboarding for small, non-production use cases via integration with
  SAFE.
- **Stateless Services in K8s** Run our stateless, supporting services
  in K8s
- **Kafka Upgrade** Upgrade Kafka to bring new features to improve
  reliability and performance. See [External Link: Kafka 2.4.1 Release
  Notes](https://archive.apache.org/dist/kafka/2.4.1/RELEASE_NOTES.html)
- **JDK 11 JARs** Provide JDK 11 only JARs to complete our move away
  from JDK 8
- **Decommission Coda** Close down vestigial use cases of Coda and
  move customers to supported alternatives, i.e Kafka.
- **Decommission Kafka Connect** migrate all use cases onto the new
  Kafka Mirrors solution.
