Choose from the following options to help you get started using ACI Kafka:

- If you’re a new customer and ready to request ACI Kafka, start with [Internal Link: On-boarding & Expansion](onboarding.html).
- If you’re trying to decide if Apache Kafka is the right choice for your service, start with the [External Link: Apache Kafka documentation](https://kafka.apache.org/documentation/#uses).
- If you’re already familiar with Apache Kafka, but you’re not sure if ACI Kafka fits your use case, review the information below on this page for [Internal Link: Offerings](#offerings) and [Internal Link: ACI Kafka caveats and limitations](#aci-kafka-caveats-and-limitations).
- If you’re just starting to develop your service, you can use the [Internal Link: Development and CI](dev-and-ci.html) resources and the client configuration [Internal Link: Quick Start](client/quick_start.html) to help you with local development.
