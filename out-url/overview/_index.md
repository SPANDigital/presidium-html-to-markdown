This **Overview** provides definitions, offerings, specific conditions, and other information about ACI Kafka services and support.

[External Link: Apache Kafka](https://kafka.apache.org/intro) is an open-source distributed event streaming platform for high-performance data pipelines, streaming analytics, data integration, and mission-critical applications.

Apple Cloud Infrastructure (ACI) Kafka offers [Internal Link: Topic-as-a-Service](#topics-as-a-service) in dedicated and shared Kafka clusters. ACI Kafka runs an up-to-date version of Kafka and provides production clusters in Apple-owned data centers (AODC), [External Link: China (GCBD)](https://kube.apple.com/docs/cross-border-connectivity-from-aodc-to-gcbd/), and AWS@Apple. A self-service API helps you manage your topics and client identities. A Java client supports cluster discovery and streamlines configuration. You can also use Topic-as-a-Service with any Kafka-compatible client.

Background information can be found in the [External Link: Apache Kafka](https://kafka.apache.org/documentation/) documentation.

Note: The term _Kafka_ refers to Apache Kafka and _ACI Kafka_ refers to offerings, tools, and clusters throughout this documentation.
