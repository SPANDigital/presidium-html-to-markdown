title: "Public API¶"
weight: 1
---

_Public API_ allows you to interact with Apple Cloud Infrastructure (ACI) Kafka entities with `curl` or a tool of your choice.
The [External Link: OpenAPI v3](https://openapi-generator.tech) lets you generate clients for several programming languages, like Java, Python, or Go, using the [External Link: API specification hosted from `pie/kafka-tools`](https://github.pie.apple.com/pie/kafka-tools/blob/develop/kafka-admin-common/src/main/resources/public_api_openapi.yml).
