weight: 1

The Kafka CLI enables users to quickly modify their entities using a command-line interface. It is built on top of the [Internal Link: public API](public_api.html) using [External Link: OpenAPI Generator](https://openapi-generator.tech). The CLI is published under the [External Link: Kafka CLI package](https://engweb.apple.com/pypi/pie-kafka-cli) with its [External Link: source available on PIE GitHub](https://github.pie.apple.com/pie/kafka-cli). The Python client requires Python 3.7 or later.
