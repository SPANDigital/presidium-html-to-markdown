You can download the [External Link: producer dashboard
template](https://github.pie.apple.com/pie/kafka-tools/tree/develop/kafka-hubble-reporter/src/main/templates/hubble-producer-dashboard.json)
and [External Link: consumer dashboard
template](https://github.pie.apple.com/pie/kafka-tools/tree/develop/kafka-hubble-reporter/src/main/templates/hubble-consumer-dashboard.json)
from GitHub. Once you’ve downloaded the templates, substitute your
Hubble Application Name where you see the
_{hubble\_app\_name}_ token and then apply the JSON in your
Edit Hubble Dashboard View.
