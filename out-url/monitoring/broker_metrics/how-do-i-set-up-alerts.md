title: "How do I set up alerts?¶"
weight: 7
---

To set up alerting, you will need to:

1. On-board with [External Link: ASE Mosaic](https://telemetry.g.apple.com).
2. Once you have your own Mosaic workspace, contact the ASE Kafka team and we will share the metrics with your workspace.
3. Once this is set up, follow [External Link: Mosaic’s alerting documentation](https://telemetry.g.apple.com/docs/alerting).
   If you have any other questions please contact the ASE Mosaic team directly on Slack in
   [External Link: `#help-ase-mosaic`](https://a1391190.slack.com/messages/CHNFM578E).
