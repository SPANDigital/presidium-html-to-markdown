Releases¶

Recommended release¶

The current production recommended release is 0.29.

Announcement Procedure¶

ACI Kafka release announcements are sent to aci-kafka-announce and kafka-users. Announcements include new features, bug fixes, improvements, highlights and API and configuration changes in the ACI Kafka client.
