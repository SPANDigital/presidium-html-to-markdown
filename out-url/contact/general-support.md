We are active on [External Link: Stack Overflow](https://stackoverflow.apple.com) using
the tag
[External Link: aci-kafka](https://stackoverflow.apple.com/questions/tagged/aci-kafka).
It’s our preferred way to provide support for questions that can be
asked and answered in an open forum—so as to benefit others.

For questions which are [External Link: not a good fit for Stack
Overflow](https://stackoverflow.apple.com/help/dont-ask), contact
[Internal Link: aci-queue-support@group.apple.com](mailto:aci-queue-support%40group.apple.com).

If you need specific support for a non-critical issue please file a
radar against [External Link: ASE Queue \| Kafka
Support](rdar://new/problem?componentid=988487).

When contacting us please provide as much information as you can,
omitting anything confidential. Consider including:

- Some context on what you’re trying to achieve
- A brief explanation of the impact (e.g. development blocker,
  performance degradation, etc)
- Details of what you did
- What you expected to happen
- What actually happened
- Details of the client library you are using
- Details of the client configuration you are using - including your
  customer group information.
- Detailed logs - ideally at debug level
- Tell us what you have what you have already tried any theories you
  have about the nature of the problem.
