This section of the documentation explains how to use the set of
extensions we’ve provided for the stock kafka client.

- An easy to extend and lightweight envelope
- Safe defaults which favor consistency rather than throughput
- Integration with the rest of the ACI Kafka environment
- Provide a level of indirection for clusters by replacing them with
  the namespace concept
