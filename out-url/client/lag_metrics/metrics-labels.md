weight: 2

Metrics available to consumer groups are enumerated below.

| Metric name | Description |
| --- | --- |
| `kafka_consumergroup_group_lag` | The difference between the last produced offset and the last consumed offset for this partition in this topic partition for this consumer group. |
| `kafka_consumergroup_group_lag_seconds` | A rough estimate of the lag in seconds. This metric can be misleading if the traffic flow is inconsistent or spiky. |
| `kafka_consumergroup_group_offset` | The last consumed offset for this partition in this topic partition for this consumer group. |
| `kafka_consumergroup_group_topic_sum_lag` | The sum of differences between the last produced offset and the last consumed offset of all partitions in this topic for this consumer group. Including idle consumer instances. |
| `kafka_consumergroup_group_sum_lag` | The sum of the differences between the last produced offset and the last consumed offset of all partitions for this consumer group. Including idle consumer instances. |

### Labels[Internal Link: ¶](\#labels)

#### Common labels[Internal Link: ¶](\#common-labels)

- `_ws_` : Mosaic workspace. This is a constant value set to `aci-kafka` and cannot be changed.
- `_ns_`: Mosaic namespace. Either `kafka-consumer-lag-prod` or `kafka-consumer-lag-if1`.
- `cluster_name`: Kafka cluster name.

#### Consumer group labels[Internal Link: ¶](\#consumer-group-labels)

- `consumer_group`: The value of [Internal Link: `group.id`](consumer_client.html#consumer-group-a-k-a-group-id-config)
- `client`: ACI Kafka [Internal Link: client identity](../concepts.html#client-identities). This value is extracted from `consumer_group`.
- `client_customer_group`: ACI Kafka [Internal Link: customer group](../concepts.html#customer-group) of the consumer group.

Additionally, the metrics that are not summed include the following label:

- `consumer_id`: The id of the consumer group member. Usually, this is automatically generated by the group coordinator. If the instance has been idle for 7 days, then `consumer_id` will be `unknown`.

#### Topic labels[Internal Link: ¶](\#topic-labels)

- `namespace`: ACI Kafka [Internal Link: namespace](../concepts.html#namespace) of the topic.
- `customer_group`: ACI Kafka [Internal Link: customer group](../concepts.html#customer-group) of the topic.
- `topic_name`: ACI Kafka topic name. This is **not** the fully qualified name.
- `partition`: The Kafka partition.
