This quick start guide provides an overview of the configuration needed
for ACI Kafka clients and example code for instantiating a producer and
consumer client. Further explanation of [Internal Link: ACI Kafka
Concepts](../concepts.html) and [Internal Link: Kaffe](kaffe.html) can be found in
other sections of our documentation.

This quick start guide is written for Java (or JVM)-based services. For
non-Java services, see [Internal Link: Kafka for All (non-Java
clients)](kafka_for_all.html).
